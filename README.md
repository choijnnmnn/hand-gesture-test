# Hand Gesture Test - MediaPipe Index Finger Tracking & Korean Sign Language Learning

Google MediaPipe를 사용한 손가락 추적 및 한글 수화 학습 웹 애플리케이션입니다.

## 🚀 주요 기능

### 📹 **기본 손가락 추적 기능**
- **실시간 손 추적**: 웹캠을 통한 실시간 손 인식
- **Index Finger Tip 강조**: 검지 끝부분을 빨간색으로 특별 표시
- **랜드마크 시각화**: 모든 손가락 관절을 색상별로 표시
- **연결선 표시**: 손가락과 손바닥의 연결선을 표시
- **실시간 좌표**: 검지 관련 관절들의 3D 좌표 실시간 표시
- **다중 손 인식**: 최대 2개의 손을 동시에 인식
- **반응형 디자인**: 모바일과 데스크톱 모두 지원

### 🤚 **한글 수화 학습 시스템** (NEW!)
- **이름 입력 및 분석**: 한글 이름을 자모음으로 분해
- **자모음별 수화 학습**: 초성, 중성, 종성별 수화 방법 학습
- **실시간 수화 인식**: 손가락 패턴을 실시간으로 인식하고 정확도 측정
- **단계별 학습**: 자모음 → 음절 → 전체 이름 순서로 학습
- **피드백 시스템**: 즉시 시각적 피드백 제공
- **진행도 추적**: 학습 진행률 및 통계 실시간 표시
- **성취 시스템**: 배지 및 점수 시스템으로 동기부여
- **게임화 요소**: 점수, 배지, 랭킹 시스템

## 📋 사용 방법

### 기본 손가락 추적 (`index.html`)
1. **웹 브라우저에서 `index.html` 파일을 열기**
2. **카메라 권한 허용**: 브라우저에서 카메라 접근 권한을 허용
3. **카메라 시작**: "카메라 시작" 버튼 클릭
4. **손 보여주기**: 카메라 앞에 손을 보여주면 자동으로 인식
5. **결과 확인**: 
   - 빨간색 점: 검지 끝부분 (Index Finger Tip)
   - 파란색 점: 다른 손가락 관절들
   - 녹색 선: 손가락과 손바닥 연결선
   - 실시간 좌표 정보 업데이트

### 한글 수화 학습 (`korean-sign-learning.html`)
1. **웹 브라우저에서 `korean-sign-learning.html` 파일을 열기**
2. **이름 입력**: 자신의 한글 이름을 입력 (예: 김철수)
3. **이름 분석**: "이름 분석하기" 버튼 클릭
4. **카메라 시작**: "카메라 시작" 버튼 클릭
5. **연습 모드**: "연습 모드" 버튼 클릭
6. **수화 학습**: 화면의 안내에 따라 자모음별 수화 연습
7. **진행 확인**: 실시간 피드백과 진행도 확인

## 🎯 MediaPipe Hand Landmarks

MediaPipe는 각 손에 대해 21개의 랜드마크를 제공합니다:

### 검지 (Index Finger) 관련 랜드마크:
- **5번**: Index Finger MCP (Metacarpophalangeal joint)
- **6번**: Index Finger PIP (Proximal Interphalangeal joint)  
- **7번**: Index Finger DIP (Distal Interphalangeal joint)
- **8번**: Index Finger Tip (검지 끝부분) ⭐

### 다른 손가락들:
- **0-4번**: 엄지 (Thumb)
- **9-12번**: 중지 (Middle Finger)
- **13-16번**: 약지 (Ring Finger)
- **17-20번**: 소지 (Pinky)

## 🤚 한글 수화 패턴 예시

### 초성 패턴 (일부)
- **ㄱ**: 엄지와 검지를 펴고 나머지는 구부리기
- **ㄴ**: 검지와 중지를 펴고 나머지는 구부리기
- **ㄷ**: 검지만 펴고 나머지는 구부리기
- **ㅁ**: 소지만 펴고 나머지는 구부리기
- **ㅂ**: 엄지, 검지, 중지를 펴고 나머지는 구부리기

### 중성 패턴 (일부)
- **ㅏ**: 엄지와 검지를 펴고 나머지는 구부리기
- **ㅓ**: 검지만 펴고 나머지는 구부리기
- **ㅗ**: 엄지만 펴고 나머지는 구부리기
- **ㅜ**: 소지만 펴고 나머지는 구부리기
- **ㅣ**: 엄지만 구부리고 나머지는 펴기

## 🔧 설정 옵션

MediaPipe Hands 설정을 변경하려면 `initializeMediaPipe()` 함수에서 다음 옵션들을 조정할 수 있습니다:

```javascript
this.hands.setOptions({
    maxNumHands: 1,              // 최대 손 개수 (수화 학습용)
    modelComplexity: 1,          // 모델 복잡도 (0-1)
    minDetectionConfidence: 0.7, // 최소 감지 신뢰도
    minTrackingConfidence: 0.7   // 최소 추적 신뢰도
});
```

## 📊 좌표 시스템

- **X축**: 0 (왼쪽) ~ 1 (오른쪽)
- **Y축**: 0 (위쪽) ~ 1 (아래쪽)  
- **Z축**: 손목에서 멀어질수록 양수값

## 🚨 문제 해결

### 카메라가 작동하지 않는 경우:
1. 브라우저에서 카메라 권한이 허용되었는지 확인
2. 다른 애플리케이션에서 카메라를 사용 중인지 확인
3. HTTPS 환경에서 실행하고 있는지 확인

### 손이 인식되지 않는 경우:
1. 충분한 조명 확보
2. 카메라와 손 사이의 적절한 거리 유지
3. 손이 카메라 화면에 완전히 들어오는지 확인

### 수화 인식이 부정확한 경우:
1. 수화 패턴의 임계값 조정 (현재 80%)
2. 손가락 상태 판단 기준 조정
3. 연습 모드에서 반복 연습
